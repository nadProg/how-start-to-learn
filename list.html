Книги

  • expand_moreJavaScript для детей

    В книге при помощи библиотеки jQuery показываются первые шаги в написании веб-приложений. Первые скрипты, работа с графикой и анимацией, инструменты разработчика. Если вы никогда раньше не сталкивались с веб-разработкой, не стесняйтесь названия и начните с этой книги.

  • expand_moreГрокаем алгоритмы

    По аналогии эту книгу можно назвать "Алгоритмы для детей". Очень простым языком объясняются базовые структуры данных, алгоритмы, включая оценку их сложности. После ее прочтения вы легко сможете сравнить производительность линейного и бинарного поиска с соответствии с О‑нотацией, даже если не данный момент это вам абсолютно ни о чем не говорит.
    open_in_new

  • expand_moreРефакторинг и Паттерны

    Эта книга тех, кто уже уверенно пишет простые скрипты на JS, но не понимает как перейти к более объемным приложениям. Реализация и поддержка сложных проектов требует соблюдения определенных принципов и применения определенных решений. К счастью часто не нужно изобретать велосипед и достаточно правильно применять проверенные принципы и паттерны проектирования.

Он-лайн учебники, справочники

  • expand_moreLearn Javascript

    На мой взгляд это самый лучший учебник для изучения JS на русском языке. Является качественным переводом оригинального ресурса javascript.info. Информация хорошо структурирована, много примеров, есть задачи для практики. Регулярно обновляется.
    open_in_new

  • expand_moreMDN Web Docs

    Самый актуальный справочник по базовым технологиям веб-разработки. Лучше пользоваться английской версией, т.к. переводов на русский мало, а их корректность не проверена.
    open_in_new

  • expand_moreW3Schools

    Сборник туториалов и справочников по веб-разработке. Информация краткая, но по всем темам есть песочница с примерами. Можно пользоваться как интерактивным справочником по различным веб-технологиям.
    open_in_new

  • expand_moreMETANIT

    Сайт о программировании, в том числе есть раздел по веб-разработке. Материалы изложены в формате учебника с примерами.
    open_in_new

Тренажеры

  • expand_moreFreeCodeCamp

    Сборник тренажеров по фронтенду начиная от HTML/CSS и заканчивая React-Redux. Есть песочница для проверки алгоритмических задач из Проекта Эйлера
    open_in_new

  • expand_moreHTMLAcademy

    Очень много интересных тренажеров по базовым технологиям. К сожалению доступ платный, но цена приемлемая.
    open_in_new

  • expand_moreCodesignal

    Площадка для решения алгоритмических задач. Можно использовать разные языки, в том числе JS. Задачи структурированы по сложности и по темам. Отсутствие подписки и виртуальной валюты не блокирует доступ к ресурсу, а только лимитирует порядок доступа к задачам и подсказки.
    open_in_new

  • expand_moreCodewars

    Площадка для решения алгоритмических задач. Можно использовать разные языки, в том числе JS. Задачи распределены по уровням. Можно прокачивать свой уровень решая все более сложные задачи. Ограничений на выбор задач нет.
    open_in_new
